# 漏斗分析模板

## 漏斗分析使用说明

漏斗分析用于分析多步骤转化链路中，用户在哪个环节流失最多。

### 适用场景
- 注册转化率低
- 付费转化率低
- 功能使用率低
- 任何多步骤流程的转化分析

### 关键定义
- **转化率**：当前层人数 / 上一层人数
- **累计转化率**：当前层人数 / 第一层人数
- **流失率**：1 - 转化率
- **时间窗口**：用户在多长时间内完成这些步骤才算（如 7 天内 / 单次会话内）
