# AI PM SkillKit — AI 产品经理方法论工具包

> 8 个开箱即用的 AI 产品经理方法论 Skill，覆盖用户研究、产品设计、数据分析、求职发展四大类。每个 Skill 包含完整工作流、输出模板、提示词合集和实战案例。

---

## 这是什么

AI PM SkillKit 是一套为 AI 产品经理打造的方法论工具包。每个 Skill 都是一个独立的、可直接调用的工作流——告诉你**遇到什么问题用什么方法、每一步做什么、产出什么**，并附带可直接套用的模板和提示词。

- **系统化**：不是零散的技巧，是完整的 5-6 步工作流
- **可落地**：每个方法都有模板和示例，拿来就能用
- **AI 原生**：针对 AI 产品的特点设计（如 AI 需求分析、AI 参与模式等）
- **即开即用**：打包为 `.skill` 文件，可直接安装到 Aily 等支持 Skill 的平台

## 安装与使用

### 方式一：直接安装 .skill 文件（推荐）

1. 从 [`dist/`](dist/) 目录下载需要的 `.skill` 文件
2. 在支持 Skill 的平台（如 Aily）中导入安装
3. 直接用自然语言触发，例如：
   - "帮我做一份飞书和钉钉的竞品调研"
   - "优化一下我的简历"
   - "DAU 跌了，帮我分析一下原因"

### 方式二：源码使用

每个 Skill 的源码都在 [`skills/`](skills/) 目录下，你可以：

- 直接阅读 `SKILL.md` 学习方法论
- 复制 `references/` 中的模板到自己的工作流中
- 根据需要自定义修改后重新打包

### 重新打包 Skill

```bash
cd skills/
zip -r ../dist/skill-name.skill skill-name/
```

## 每个 Skill 的结构

每个 Skill 遵循统一的目录结构，方便理解和扩展：

```
skill-name/
├── SKILL.md              # 主文档：触发条件 + 工作流 + 参考索引
└── references/
    ├── template-xxx.md   # 输出物模板（2-4 个）
    ├── prompts.md        # 提示词合集（各阶段可直接使用的 Prompt）
    └── examples.md       # 实战案例（完整示例参考）
```
